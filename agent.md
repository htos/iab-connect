Titel
Agent Instructions for IAB Connect

Regel
1) Lies immer zuerst docs/00_agent_context.md
2) Halte dich strikt an die Templates und Regeln dort
3) Ändere oder ergänze nur Dateien unter docs
4) Wenn etwas unklar ist, triff eine Annahme und schreibe sie in docs/09_decisions_log.md
5) Requirements müssen immer im Format Titel, Beschreibung, Funktionen, Akzeptanzkriterien sein, ohne Formatierungshilfen
6) Requirements Inhalte kommen aus docs/Anforderungen_WebApp_Indischer_Kulturverein.csv
7) Status wird in docs/10_requirements_status.md geführt
8) Matching erfolgt immer über die Spalte ID aus der CSV
9) Erlaubte Status Werte sind Backlog, Ready, InProgress, Blocked, Done, Dropped
10) CSV Inhalte werden nie geändert ausser der User verlangt es explizit
